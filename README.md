# Production Management Dashboard

A web-based internal dashboard designed to streamline production operations, employee management, and planning workflows.

## Overview

This project is a centralized production management system built to replace manual and spreadsheet-based workflows.

It is designed to support multiple operational features across the production pipeline, including employee management, production planning, and logistics.

Currently, the Daily Lineup feature is actively being developed, while other modules are under construction.

## Demo

Live demo: https://johnstons-dashboard.vercel.app/assignment-board

## Current Status

- Daily Lineup: In progress  
- Other features: Under construction  

## Available Feature

### Daily Lineup

The Daily Lineup feature focuses on managing daily employee assignments and availability.

It allows supervisors to:

- Track employee attendance (active, absent, sick, vacation, injured, training)
- View employees by department
- Assign employees to stations
- Manage department-level staffing
- Prepare shift-based lineups
- Display the lineup on a TV-friendly screen for the production floor
